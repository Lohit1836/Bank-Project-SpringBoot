# 🏦 Bank Project Using Spring Boot

A secure and efficient **Bank Management Application** developed using **Spring Boot**, **Spring Security**, **Spring Data JPA**, and **MySQL**.  
This project demonstrates backend development, authentication/authorization, and REST API design for managing bank operations.

---

##  Tech Stack
- **Backend:** Java, Spring Boot, Spring Security, Spring Data JPA
- **Database:** MySQL
- **Tools:** Maven, Postman, Git
- **Architecture:** MVC Pattern

---

##  Features
- User authentication and role-based authorization using **Spring Security**
- Account creation, balance check, and transaction management APIs
- **CORS** and **CSRF protection** implemented for enhanced security
- RESTful API endpoints tested using **Postman**
- Follows clean code practices and modular design structure
